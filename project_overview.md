# Обзор проекта

Этот документ представляет собой краткий обзор проекта, основанный на анализе кодовой базы.

## Технологии

*   **Фреймворк:** Next.js (с App Router)
*   **Язык:** TypeScript
*   **База данных:** MySQL с Prisma ORM
*   **Аутентификация:** NextAuth.js
*   **Стилизация:** Tailwind CSS
*   **UI-компоненты:** shadcn/ui
*   **Развертывание:** Firebase Hosting (судя по `apphosting.yaml`)
*   **Дополнительно:** Возможно, есть мобильное приложение на React Native.

## Структура проекта

*   **`src/app`**: Содержит страницы приложения, созданные с помощью Next.js App Router. Основные страницы:
    *   `login`: Аутентификация пользователя.
    *   `ranking`: Отображение рейтинга спортсменов.
    *   `tournament`: Детали турнира и, возможно, регистрация.
    *   `certification`: Информация об аттестациях.
    *   `cert`: Просмотр отдельных сертификатов.
*   **`src/components`**: Хранит переиспользуемые React-компоненты, используемые в приложении.
*   **`src/lib`**: Содержит бизнес-логику, взаимодействие с базой данных (`db.ts`) и логику API (`fighters-api.ts`, `tournaments-api.ts`).
*   **`prisma`**: Определяет схему базы данных (`schema.prisma`).
*   **`mobile`**: Предполагает наличие мобильного приложения на React Native.
*   **`docs/blueprint.md`**: Вероятно, содержит архитектурную документацию.

## Основной функционал

Проект представляет собой платформу для управления мероприятиями и сертификациями в области единоборств. Судя по коду, ключевые функции включают:

*   **Аутентификация пользователей:** Пользователи могут входить в систему.
*   **Управление мероприятиями:** Платформа управляет турнирами и аттестациями.
*   **Рейтинги спортсменов:** Существует система для ранжирования спортсменов.
*   **Генерация и проверка сертификатов:** Система может генерировать PDF-файлы для сертификатов, включая QR-коды для проверки (как видно в `CertificationPDF.tsx`).
*   **Система разрядов:** Реализована система присвоения и отслеживания спортивных разрядов/уровней (как видно в `RazryadSystem.tsx`).
